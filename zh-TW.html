關於 Rocky's Studio

Rocky's Studio 是一組精選的免費瀏覽器端開發者工具，旨在讓日常開發任務更快速、更便捷。無論你需要格式化和驗證JSON、測試正規表示式、查看X.509憑證，還是分析網路封包擷取，Rocky's Studio 都能滿足你的需求。

所有工具均完全在瀏覽器中執行，資料絕不會離開你的裝置。無需註冊、無Cookie、無追蹤。這是雜亂線上工具網站的隱私優先替代方案 — 快速、專注、永久免費。

本頁面是 Rocky's Studio 的官方介紹和討論中心。你可以在這裡討論功能、提議新工具、回報Bug，以及關注開發更新。

33+ 免費開發者工具

Rocky's Studio 提供的完整免費線上開發者工具列表。點擊任意工具即可在主站打開。

JSON 格式化

即時語法高亮和錯誤診斷，格式化、驗證和美觀化JSON。

JSON Schema 驗證

根據JSON Schema驗證JSON文件，提供詳細且可執行的錯誤訊息。

URL 編解碼

在瀏覽器中即時編解碼URL百分號編碼（RFC 3986）。

Base64 轉換

將文字或二進位資料編碼為Base64，或將Base64解碼回純文字 — 完全離線執行。

正規測試

建構、測試和除錯正規表示式，在多個測試字串上即時高亮匹配結果。

顏色選擇器

視覺化選擇顏色，支援HEX、RGB、HSL、HSV、CMYK和CSS顏色格式之間的轉換。

HTTP 用戶端

直接從瀏覽器發送HTTP請求（GET、POST、PUT、DELETE、PATCH），完整查看請求和回應。

時間戳轉換

在Unix時間戳和人類可讀日期之間相互轉換。支援秒、毫秒和微秒。

UUID 生成器

即時生成隨機UUID v4識別碼。一鍵批次生成多個UUID。

SQLite 編輯器

直接在瀏覽器中讀取、查詢和編輯SQLite資料庫（.db、.sqlite）— 無需伺服器。

單位換算

轉換長度、重量、體積、面積、時間、溫度等多種單位 — 在瀏覽器中快速免費使用。

線上計算機

功能齊全的計算機，支援標準型、科學型和程式設計師模式 — 包含二進位、十六進位、八進位轉換。

密碼生成器

生成密碼學安全的隨機密碼，可自訂長度、字元集和複雜度規則。

X.509 憑證檢視器

解碼、檢視和分析PEM或DER格式的X.509 SSL/TLS憑證 — 查看主旨、簽發者、擴充功能和有效期。

憑證鏈建構

建構、驗證和校驗X.509憑證鏈，支援簽章驗證、DN連結和鏈順序檢查。

封包分析器 (PCAP)

解析和檢視.pcap和.pcapng網路擷取檔案 — 支援Ethernet、IPv4、IPv6、TCP、UDP、ICMP、DNS、DHCP、HTTP和TLS層。

壓縮檔檢視器

在瀏覽器中檢視ZIP、TAR、GZIP、.tar.gz、.7z、.zst和.tar.zst壓縮檔，支援CSV/JSON/圖片預覽和巢狀壓縮檔深入檢視。

NMEA 0183 解碼器

解碼GNSS NMEA訊句（GGA、RMC、GSA、GSV、VTG、GLL、ZDA、TXT），支援校驗和驗證和可見衛星彙總。

QR碼生成器

從文字或URL生成自訂QR碼，可設定糾錯等級、像素大小和顏色。支援PNG下載。

線上文字編輯器

功能強大的瀏覽器端文字編輯器，支援正規尋找取代、大小寫轉換、行排序、自動儲存草稿和檔案下載。

Markdown 預覽

即時Markdown轉HTML轉換器，支援GitHub Flavored Markdown (GFM)、語法高亮和複製到剪貼簿。

資料夾/檔案比對

並排比較兩個文字檔或兩個資料夾樹，支援行級差異高亮和忽略空白字元選項。

ASCII 藝術生成器

將任意文字轉換為裝飾性ASCII藝術橫幅 — 支援Banner、Block、Bold、Shadow、3D、Graffiti、Bubble、Matrix和Slant風格。

時區轉換

世界時鐘、時區瀏覽器和轉換器 — 查看主要城市的本地時間，在IANA時區之間轉換，瀏覽日光節約時間資訊。

雜湊計算器 (MD5 / SHA)

為文字或檔案計算MD5、SHA-1、SHA-256、SHA-384和SHA-512雜湊 — 完全在瀏覽器中執行，支援可選HMAC。

JWT 工具

解碼、檢視、編碼和簽署JSON Web Tokens (JWT) — 支援HS256/384/512、RS256/384/512、PS256/384/512和ES256/384/512演算法。

圖片壓縮

在瀏覽器中批次壓縮JPEG、PNG、WebP和AVIF圖片 — 調整品質和解析度，然後單獨下載或打包為ZIP。

SVG 轉圖片

將SVG向量圖形轉換為PNG、JPEG或WebP — 自訂尺寸和背景色，還可生成多尺寸favicon ZIP包。

圖片編輯

批次調整大小、裁切、加入浮水印和重新格式化圖片 — JPEG、PNG、WebP、AVIF — 全部在瀏覽器中完成。

PDF 工具集

一體化PDF工具集 — 合併、分割、壓縮、圖片互轉、加密解密、擷取文字。所有處理均在瀏覽器本機完成。

CSV 工具集

解析CSV為JSON，轉換JSON為CSV，支援排序查看表格資料。

資料生成器

為測試和開發生成模擬資料，可自訂欄位和格式。

UML 圖表

建立專業UML圖表，包括流程圖、類別圖、序列圖等。
