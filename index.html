About

Rocky's Studio is a collection of small, focused web tools that run entirely in your browser. No sign-up, no uploads to a server, no tracking — just fast utilities for everyday developer and power-user tasks.

This repository is the introduction and discussion hub for the project. Use it to propose features, report bugs, share ideas, and follow development updates.

Tools

A quick overview of what's available in the studio.

JSON Formatter

Format, validate, and beautify JSON with real-time preview.

JSON Schema Validator

Validate JSON against a schema with detailed diagnostics.

URL Encoder

Encode and decode URLs instantly.

Base64 Converter

Encode and decode Base64 strings.

Regex Tester

Test and debug regular expressions with match highlighting.

Color Picker

Convert between color formats (HEX, RGB, HSL, HSV).

HTTP Client

Send HTTP requests and view responses in-browser.

Timestamp Converter

Convert between Unix timestamps and human-readable dates.

UUID Generator

Generate random UUID v4 identifiers.

SQLite Editor

Read, edit and query SQLite databases directly in your browser.

Unit Converter

Convert between length, weight, volume, area, time and temperature.

Online Calculator

Simple, Scientific and Programmer modes in one calculator.

Password Generator

Generate secure random passwords with customizable options.

X.509 Certificate Viewer

Decode, inspect, and analyze X.509 certificates.

Certificate Chain Builder

Build and validate X.509 certificate chains with signature verification.

Packet Analyzer

Parse .pcap / .pcapng captures — Ethernet, IPv4/IPv6, TCP, UDP and more.

Compressed File Viewer

Inspect ZIP, TAR, GZIP, .tar.gz, .7z, .zst archives with preview.

NMEA Decoder

Decode GNSS NMEA 0183 sentences with human-readable fields.

QR Code Generator

Custom QR codes with configurable error correction and colors.

Text Editor

Rich in-browser text editor with regex find/replace and draft save.

Markdown Preview

Live markdown to HTML converter.

Folder / File Diff

Compare two text files or two folder trees side-by-side.

ASCII Art Generator

Turn text into beautiful ASCII-art banners in many styles.

Timezone Converter

World clock and IANA timezone explorer with DST info.

How to contribute

  1. Check existing discussions and issues before opening a new one.
  2. Use the Q&A category for questions, Ideas for feature proposals.
  3. When reporting a bug, include steps to reproduce, browser, and a screenshot when helpful.
  4. Be kind — this project is maintained by volunteers.
