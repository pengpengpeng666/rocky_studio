About Rocky's Studio

Rocky's Studio is a curated collection of free, browser-based developer tools designed to make everyday tasks faster and easier. Whether you need to format and validate JSON, test regular expressions, inspect X.509 certificates, or analyze network packet captures, Rocky's Studio has a tool for it.

Every tool runs entirely in your browser. Zero data ever leaves your device. No account required, no cookies, no tracking. It's the privacy-first alternative to cluttered online tool sites — fast, focused, and free forever.

This repository is the official introduction and discussion hub for Rocky's Studio. Use it to report bugs, suggest new tools, discuss features, and follow the project's development.

Free Developer Tools — 24+ In-Browser Utilities

Browse the complete suite of Rocky's Studio developer tools. All tools are free, run in your browser, and process data locally.

JSON Formatter

Format, validate, and beautify JSON with real-time syntax highlighting and error diagnostics.

JSON Schema Validator

Validate JSON documents against a JSON Schema with detailed, actionable error messages.

URL Encoder / Decoder

Encode and decode URL percent-encoding (RFC 3986) instantly in your browser.

Base64 Encoder / Decoder

Encode text or binary data to Base64 and decode Base64 back to plain text — fully offline.

Regex Tester

Build, test, and debug regular expressions with live match highlighting across multiple test strings.

Color Picker

Pick colors visually and convert between HEX, RGB, HSL, HSV, CMYK, and CSS color formats.

HTTP Client

Send HTTP requests (GET, POST, PUT, DELETE, PATCH) directly from your browser with full request/response inspection.

Timestamp Converter

Convert Unix timestamps to human-readable dates and vice versa. Supports seconds, milliseconds, and microseconds.

UUID Generator

Generate random UUID v4 identifiers instantly. Batch-generate multiple UUIDs with one click.

SQLite Editor

Read, query, and edit SQLite databases (.db, .sqlite) directly in your browser — no server required.

Unit Converter

Convert length, weight, volume, area, time, temperature and more — fast and free in your browser.

Online Calculator

A full-featured calculator with Simple, Scientific, and Programmer modes — binary, hex, octal conversions included.

Password Generator

Generate cryptographically secure random passwords with customizable length, character sets, and complexity rules.

X.509 Certificate Viewer

Decode, inspect, and analyze X.509 SSL/TLS certificates in PEM or DER format — view subject, issuer, extensions, and validity.

Certificate Chain Builder

Build, validate, and verify X.509 certificate chains with signature verification, DN linking, and chain ordering checks.

Packet Analyzer (PCAP)

Parse and inspect .pcap and .pcapng network capture files — supports Ethernet, IPv4, IPv6, TCP, UDP, ICMP, DNS, DHCP, HTTP, and TLS layers.

Compressed File Viewer

Inspect ZIP, TAR, GZIP, .tar.gz, .7z, .zst, and .tar.zst archives in-browser with CSV/JSON/image preview and nested archive drill-down.

NMEA 0183 Decoder

Decode GNSS NMEA sentences (GGA, RMC, GSA, GSV, VTG, GLL, ZDA, TXT) with checksum validation and satellite-in-view summary.

QR Code Generator

Generate custom QR codes from text or URLs with configurable error correction level, pixel size, and colors. Download as PNG.

Text Editor

A powerful in-browser text editor with regex find-and-replace, case transforms, line sorting, auto-save drafts, and file download.

Markdown Preview

Live markdown-to-HTML converter with GitHub Flavored Markdown (GFM) support, syntax highlighting, and copy-to-clipboard.

Folder / File Diff

Compare two text files or two folder trees side-by-side with line-level diff highlighting and an ignore-whitespace option.

ASCII Art Generator

Turn any text into decorative ASCII-art banners — supports Banner, Block, Bold, Shadow, 3D, Graffiti, Bubble, Matrix, and Slant styles.

Timezone Converter

World clock, timezone explorer, and converter — see local time in major cities, convert between IANA timezones, and browse DST information.

How to contribute

  1. Check existing discussions before opening a new one.
  2. Use a clear, descriptive title that summarises your question, idea, or bug.
  3. When reporting a bug, include steps to reproduce, your browser and OS, and a screenshot when helpful.
  4. Be kind — this project is maintained by volunteers.
