Acerca de Rocky's Studio

Rocky's Studio es una colección seleccionada de herramientas de desarrollo gratuitas basadas en navegador diseñadas para hacer las tareas diarias más rápidas y fáciles. Ya sea que necesites formatear y validar JSON, probar expresiones regulares, inspeccionar certificados X.509 o analizar capturas de paquetes de red, Rocky's Studio tiene una herramienta para ello.

Cada herramienta se ejecuta completamente en tu navegador. Ningún dato sale de tu dispositivo. No requiere registro, sin cookies, sin seguimiento. Es la alternativa centrada en la privacidad a los sitios de herramientas online desordenados — rápida, enfocada y gratuita para siempre.

Esta página es el centro oficial de introducción y discusión de Rocky's Studio. Úsala para discutir funcionalidades, proponer nuevas herramientas, reportar errores y seguir las actualizaciones de desarrollo.

JSON Formatter

Format, validate, and beautify JSON with real-time syntax highlighting and error diagnostics.

JSON Schema Validator

Validate JSON documents against a JSON Schema with detailed, actionable error messages.

Encode and decode URL percent-encoding (RFC 3986) instantly in your browser.

Base64 Converter

Encode text or binary data to Base64 and decode Base64 back to plain text — fully offline.

Regex Tester

Color Picker

HTTP Client

Timestamp Converter

Convert Unix timestamps to human-readable dates and vice versa. Supports seconds, milliseconds, and microseconds.

UUID Generator

Generate random UUID v4 identifiers instantly. Batch-generate multiple UUIDs with one click.

SQLite Editor

Read, query, and edit SQLite databases (.db, .sqlite) directly in your browser — no server required.

Unit Converter

Convert length, weight, volume, area, time, temperature and more — fast and free in your browser.

Online Calculator

A full-featured calculator with Simple, Scientific, and Programmer modes — binary, hex, octal conversions included.

Password Generator

Generate cryptographically secure random passwords with customizable length, character sets, and complexity rules.

X.509 Certificate Viewer

Packet Analyzer (PCAP)

Parse and inspect .pcap and .pcapng network capture files — supports Ethernet, IPv4, IPv6, TCP, UDP, ICMP, DNS, DHCP, HTTP, and TLS layers.

Compressed File Viewer

Folder / File Diff

ASCII Art Generator

Turn any text into decorative ASCII-art banners — supports Banner, Block, Bold, Shadow, 3D, Graffiti, Bubble, Matrix, and Slant styles.

Timezone Converter

World clock, timezone explorer, and converter — see local time in major cities, convert between IANA timezones, and browse DST information.

Calculate MD5, SHA-1, SHA-256, SHA-384, and SHA-512 hashes for text or files — fully in-browser with optional HMAC support.

Image Compressor

Image Editor

Batch resize, crop, add watermarks, and reformat images — JPEG, PNG, WebP, AVIF — all in your browser.

PDF Toolkit

All-in-one PDF toolkit — merge, split, compress, convert to/from images, encrypt & decrypt, and extract text. All processing happens locally in your browser.

CSV Toolkit

Data Generator

UML Diagrammer

Create professional UML diagrams including flowcharts, class diagrams, sequence diagrams, and more.
